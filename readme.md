# [Css Stats](https://cssstats.com)

Potentially interesting stats on stylesheets.

## Installation

```
git clone https://github.com/cssstats/cssstats
cd cssstats
npm i
```

## Usage


#### Development

```
npm run dev
open localhost:8000
```

#### Production

```
npm run build
npm start
```
